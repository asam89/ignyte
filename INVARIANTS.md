# INVARIANTS

Rules that must hold for all changes to the public-facing site (`public_html/`).

- Existing page structure, navigation, and routing must not change.
- Existing branding (navy `#002366`, flame orange `#EE5A24`, brand blue `#0047BB`, light grey `#F4F7FA`) must not change.
- No new dependencies (no package.json, no build tools, no external JS libraries).
- No changes to forms, contact info, or payment/contact links.
- No changes to deployment configuration (`.htaccess`, Git hooks, Hostinger settings).
